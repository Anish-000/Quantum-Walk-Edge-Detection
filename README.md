# Quantum-Walk-Edge-Detection
Extends Giri et al. (2025) quantum walk edge detection by adding a 6-direction diagonal gradient oracle and Otsu-based probability map thresholding. Evaluated on 20 BSDS500 images with NumPy statevector simulation and Qiskit AerSimulator block circuit. Achieves +4.31% Otsu Oracle F1 improvement over the baseline.
